总览

方案对比

五种方案各有侧重，可根据 program 数量和使用场景选择最适合的。也可混合使用——比如列表 + 进度环。

A. 简洁列表

水平单行，扫描效率最高
信息密度高
视觉吸引力中
适合 program 数5~20+
进度可视化进度条
移动端适配好

B. 大卡片 + 进度环

居中对称，进度环视觉焦点
信息密度低
视觉吸引力高
适合 program 数2~6
进度可视化圆环
移动端适配好

C. 紧凑网格

最小卡片面积，极致密度
信息密度最高
视觉吸引力低
适合 program 数6~30+
进度可视化进度条 + 标签
移动端适配好

D. 彩色边条

颜色区分 program，列表式布局
信息密度中高
视觉吸引力中高
适合 program 数3~15
进度可视化数字 + 进度条
移动端适配好

E. Emoji 装饰

趣味性最强，个人工具气质
信息密度中
视觉吸引力最高
适合 program 数3~9
进度可视化进度条 + 百分比
移动端适配好
