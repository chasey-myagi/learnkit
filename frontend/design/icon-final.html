设计规格
icon.svg
ViewBox: 0 0 512 512
背景: #3b82f6, rx=112 (~22%)
前景卡片: #fff, rx=20
中层卡片: rgba(255,255,255,0.30), -3.5deg
后层卡片: rgba(255,255,255,0.14), -7deg
勾号: #2563eb, stroke-width=28
文字行: #b0c4d8 / #ccd8e4
效果: 无渐变、无阴影、无发光
favicon.svg
ViewBox: 0 0 32 32
背景: #3b82f6, rx=7 (~22%)
卡片: #fff, rx=3 (简化为单张)
文字线: #c0d0e4, stroke-width=2
勾号: #2563eb, stroke-width=3
策略: 去掉堆叠，保证 16px 清晰
线条: 所有 stroke >= 2px (16px 下 >= 1px)
